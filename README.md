# Nest.js CRUD tarter kit

This kit contains the folowing features

- Typorm
- Sqlite
- Postgres
- Crud
- Pagination

# Installation

```bash
$ npm install
```

# Handle migrations

## Generate migrations

```bash
$ npm run typeorm:makemigrations "init"
```

## Run migrations

```bash
$ npm run typeorm:migrate
```

# Run

```bash
$ npm run start:dev
```

# Run with docker

```bash
$ docker-compose up build
```
